# Dashboard

This project focuses on the development of gaming dashboard using Reactjs

## Table of contents

- [Technologies](#technologies)

## Technologies

Project is created with:

- Reactjs

## Setup

To run this project:

```
* Clone the repository
* Run npm install
* npm start
```
